# 📌 Аннотация проекта

Тема проекта:
Информационная система медицинского колледжа при клинической больнице

---

## Аннотация

Разрабатываемая система представляет собой веб-приложение, предназначенное для автоматизации учебного процесса медицинского колледжа, взаимодействующего с клинической больницей, а также для управления учебными мастерскими и симуляционными кабинетами.

Цель проекта — разработать информационную систему, которая объединяет:

* учебный процесс;
* клиническую практику студентов;
* работу мастерских и симуляционных кабинетов;
* контроль состояния оборудования и его техническое обслуживание.

В системе предусмотрены следующие роли пользователей:

* администратор;
* студент;
* преподаватель;
* клинический наставник;
* техник (инженер).

Система включает несколько ключевых модулей:

* учебный модуль (группы, дисциплины, занятия, оценки, посещаемость);
* модуль практики (отделения больницы, практика, наставники);
* модуль мастерских и симуляционного центра (кабинеты, оборудование, практические занятия);
* модуль технического обслуживания (заявки на ремонт, статусы оборудования, история обслуживания).

В системе реализована регистрация и аутентификация пользователей. Часть
маршрутов защищена и доступна только авторизованным пользователям.
